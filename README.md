# Pr-traitement_d-image_textuelle
Ce programme vise à prétraiter une image textuelle en se basant sur un ensemble d’étapes principales permettant de nettoyer et d’améliorer l’image. L’objectif est d’obtenir une image prête à être analysée par un modèle OCR (Reconnaissance Optique de Caractères) en utilisant les bibliothèques OpenCV et NumPy.
